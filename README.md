# Test Report Analyzer Agent

AI-powered Test Report Analyzer using Python, Pandas and Function Calling.

## Project Goal

Analyze test execution results from CSV files and automatically generate validation statistics and reports.

## Features

* Load CSV test reports
* Validate CSV structure
* Clean and normalize data
* Calculate test statistics
* Detect failures
* Generate charts
* Generate summary reports

## Technologies

* Python
* Pandas
* Matplotlib
* GitHub

Version 1 under development.
